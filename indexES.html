Que es el Desarrollo Web?

Desarrollo Web Es el arte de poder crear lo que quieras con las herramientas adecuadas. Es el trabajo que implica el desarrollo de un sitio web para Internet (World Wide Web) o una intranet (una red privada) El desarrollo web puede abarcar desde el desarrollo de una simple página estática única de texto sin formato hasta aplicaciones web complejas, negocios electrónicos y servicios de redes sociales. Una lista más completa de tareas a las que comúnmente se refiere el desarrollo web, puede incluir ingeniería web, diseño web, desarrollo de contenido web, enlace con el cliente, secuencias de comandos del lado del cliente/del lado del servidor, Configuración de seguridad de redes y servidores web, y desarrollo de comercio electrónico.

Cuales son estas herramientas?

HTML para la estructura
CSS para el diseño
JavaScript para un sitio web interactivo.
